# search-interview
以 react + redux + redux-persist 建置搜尋課程程式

## How to use

Execute [`search-interview`](https://github.com/tony8797/search-interview) with [npm](https://docs.npmjs.com/cli/init) or [Yarn](https://yarnpkg.com/lang/en/docs/cli/create/):

```bash
npm install
npm run dev
# or
yarn 
yarn dev
```
